# Java Banking System

A simple console-based banking system built using Java.

## Features
- Show balance
- Deposit money
- Withdraw money
- Exit system

## Concepts Used
- Java
- Loops
- Conditional statements
- Scanner class

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `BankingProgram.java`
